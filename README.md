# Vef-Forritun-2

## reglum fyrir snyrtilegan kóða og vinnu rými. 

1. hafa snyrtilegan og lesilegan kóða
2. commenta vel til að gera létara fyrir aðra og þig sjálfan að lesa og breyta kóðanum
3. alltaf gera pull request og útskíra vel hvað þið breytuð þegar er að setja nýjan kóða
   

ef þið hafið einhverjar spurningar eða hugmyndir um þetta endilega talið við mig og látið mig vita.
